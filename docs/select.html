    Select


    HTML select tag is used to create drop down list of options, which appears when the user clicks on form element, and it allows to choose one of the options.



    Select Groups

    Empyreal also supports optgroups in select so you can classify select items.



    Images in Select

    You can add icons to your options in any type of select. In the option you add the image source with the data-icon attribute.
