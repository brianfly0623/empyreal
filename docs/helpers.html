    Margin and Padding

    Class Padding Right
    pr-0 0 px
    pr-1 2 px
    pr-2 4 px
    pr-3 6 px
    pr-4 8 px
    pr-5 10 px
    pr-auto auto
    As you can see there is a difference of 2px between every consecutive class. The format of these classes are: ${A}${B}-${C} where:

    `A` is either p / m which are padding and margin respectively.

    `B` is either t / b / l / r which are top, bottom, left and right respectively. This can be blank if you want the padding valve to be in all dimensions.

    `C` can be numbers between 0 to 5 or `auto` depending on the value you need.



    Flex

    Add class flex to change the display type of the element to flex and then you can use the other classes listed below.

    Flex Direction

    This class manipulates the flex-direction: style.

    It establishes the main-axis, thus defining the direction flex items are placed in the flex container.

    Class Flex Direction
    flex-row row
    flex-row-reverse row-reverse
    flex-column column
    flex-column-reverse column-reverse

    Flex Wrap

    This class manipulates the flex-warp: style.

    It defines whether the flex items are forced in a single line or can be flowed into multiple lines. If set to multiple lines, it also defines the cross-axis which determines the direction new lines are stacked in.

    Class Flex Wrap
    flex-wrap wrap
    flex-nowrap nowrap

    Align Items

    This class manipulates the align-items: style.

    The align-items property defines the default behavior for how items are laid out along the cross axis (perpendicular to the main axis).

    Class Align Items
    align-items-center center
    align-items-start flex-start
    align-items-end flex-end
    align-items-stretch stretch
    align-items-baseline baseline

    Align Self

    This class manipulates the align-self: style.

    It makes possible to override the align-items value for specific flex items.

    Class Align Self
    align-self-center center
    align-self-start flex-start
    align-self-send flex-end
    align-self-stretch stretch
    align-self-baseline baseline

    Align Content

    This class manipulates the align-content: style.

    It helps to align a flex container's lines within it when there is extra space in the cross-axis, similar to how justify-content aligns individual items within the main-axis.

    Class Align Content
    align-content-center center
    align-content-start flex-start
    align-content-end flex-end
    align-content-stretch stretch
    align-content-around around

    Justify Content

    This class manipulates the justify-content: style.

    It defines the alignment along the main axis. It helps distribute extra free space leftover when either all the flex items on a line are inflexible, or are flexible but have reached their maximum size. It also exerts some control over the alignment of items when they overflow the line.

    Class Justify Content
    justify-content-center center
    justify-content-start flex-start
    justify-content-end flex-end
    justify-content-between space-between
    justify-content-around space-around

    Order

    This class manipulates the order: style.

    Flex items are displayed in the same order as they appear in the source document by default. The order property can be used to change this ordering.

    Class Order
    order-1 1
    order-2 2
    order-3 3
    order-4 4
    order-5 5
    order-6 6
    order-7 7
    order-8 8
    order-9 9
    order-10 10
    order-11 11
    order-12 12

    Note:

    order-* class can also be used responsively by adding the screen character before the number. For example to display the item in the 3rd position in tablet devices use: order-m3



    Showing / Hiding content

    Class Screen Range
    .hide Hidden for all Devices
    .hide-on-small-only Hidden for Mobile Only
    .hide-on-med-only Hidden for Tablet Only
    .hide-on-med-and-down Hidden for Tablet and Below
    .hide-on-med-and-up Hidden for Tablet and Above
    .hide-on-large-only Hidden for Desktop Only
    .show-on-small Show for Mobile Only
    .show-on-medium Show for Tablet Only
    .show-on-large Show for Desktop Only
    .show-on-medium-and-up Show for Tablet and Above
    .show-on-medium-and-down Show for Tablet and Below
    .center-on-small-only Center text on Mobile Only
